This is for showing students how input and if-else work (explain this in class):

name = input("Enter your name: ")
marks = int(input("Enter your marks: "))
if marks >= 90:
    print("Excellent,", name)
elif marks >= 60:
    print("Good,", name)
elif marks >= 35:
    print("Pass,", name)
else:
    print("Fail,", name)
